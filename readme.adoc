== How to use the app?

- Clone the repository:
----
git clone https://github.com/Arquisoft/dechat_es1a.git
----

- Enter in the folder dechat_es1a:
----
 cd dechat_es1a
----

- Install angular dependencies:

----
npm install --save-dev @angular-devkit/build-angular
----

- Start server:

----
ng serve -o
----
